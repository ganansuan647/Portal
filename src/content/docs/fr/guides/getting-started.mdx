## 配置说明

<Steps>

1. <span role="img" aria-label="Unpacking Icon">📦</span> **开箱ScrewFast工具**
   小心打开包装，检查所有组件是否齐全。熟悉工具及其可能的附件。

2. <span role="img" aria-label="Safety Check Icon">🔍</span> **初始配置和安全检查**
   在操作工具前阅读安全手册。检查工具是否因运输而受损，并确保安全装置到位。

3. <span role="img" aria-label="Calibration Icon">⚙️</span> **校准及首次使用准备**
   根据制造商的说明校准您的工具（如有必要）。确保电池已充电或电连接已安全。

</Steps>

## 清洁与维护

<Aside type="tip">
  清洁前务必断开工具电源。使用柔软干燥的布擦拭外部，并将其存放在干燥处。
</Aside>

定期维护可确保工具的寿命和安全性。每次使用后清洁工具，检查磨损情况，并按照工具手册的说明润滑移动部件。
